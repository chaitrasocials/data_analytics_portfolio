# HighCloud Airlines – Data Analytics Project

This project analyzes airline performance using Excel, Power BI, and Tableau. It includes data cleaning, dashboard creation, and a final business insights presentation.

# Tools Used
	•	Excel
	•	Power BI
	•	Tableau
	•	PowerPoint

# Project Tasks
	•	Cleaned and transformed airline datasets
	•	Created dashboards in Excel, Power BI, and Tableau
	•	Analyzed KPIs like route performance, delays, revenue, and occupancy
	•	Built a final PPT summarizing insights and recommendations
